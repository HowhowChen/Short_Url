# Short_Url
# 短網址產生器

此專案能將冗長的網址縮短，方便與他人分享。

## 功能列表

- 輸入網址轉換成短網址
- 點擊copy button 複製短網址
- 透過短網址轉址到原本網址

##  安裝

1. Clone 此專案至本機電腦，打開 terminal 至欲存放專案之資料夾，輸入下列代碼

```
git clone https://github.com/HowhowChen/Short_Url.git
```

2. 進入專案資料夾，請在終端機輸入:

```
cd Short_Url
```

3. 安裝 npm 套件，請在終端機輸入：
```
npm install
```

4. 執行專案，請在終端機輸入：
```
npm run dev
```

5. 輸入下列代碼於**網址列**即可使用:
```
localhost:3000
```

6. 若欲暫停使用：
```
ctrl + c
```
